【生の時系列データ（温度など）】
 │
 ├──► 系統 A：【統計・閾値ベース】（データを綺麗に削って、分布のハミ出しを叩くルート）
 │     │
 │     ├── [ステップ1：分解・変換（前処理）]
 │     │    ├── 古典的時系列分解（移動平均でトレンド・周期を「引き算」）
 │     │    ├── STL分解（Loess回帰でトレンド・周期をロバストに「引き算」）
 │     │    └── X-11 / SEATS / X-13ARIMA-SEATS（厳密な季節調整・分解）
 │     │
 │     └── [ステップ2：判定・予測（本処理）] ── ※上記で抽出した「残差（ノイズ）」に適用
 │          ├── $3\sigma$（スリーシグマ）法
 │          ├── $X\bar{R}$管理図 / $X$管理図（工程内のバラツキ管理）
 │          ├── CUSUM（累積和）管理図（微小な変化の継続を検知）
 │          └── EWMA（指数加重移動平均）管理図（直近のデータを重視した閾値判定）
 │
 ├──► 系統 B：【予測誤差ベース】（未来を予測させて、その答え合わせのズレを見るルート）
 │     │
 │     ├── [ステップ1：分解・変換（前処理）] ── ※モデル自体が学習するため、前処理は補助的
 │     │    ├── なし（生のまま、またはラグ特徴量への変換のみ）
 │     │    └── 差分化（定常データに変換する最小限の処理）
 │     │
 │     └── [ステップ2：判定・予測（本処理）]
 │          ├── 【統計的確率モデル（線形）】
 │          │    ├── AR（自己回帰）
 │          │    ├── ARIMA（自己回帰和分移動平均）
 │          │    ├── SARIMA（季節自己回帰和分移動平均）
 │          │    ├── GARCH（ボラティリティ・分散の変動を予測）
 │          │    └── 状態空間モデル / カルマンフィルタ（ノイズから真の値を推定）
 │          │
 │          └── 【機械学習・ディープラーニングモデル（非線形・複雑）】
 │               ├── Prophet（Meta製・トレンドと周期の一般化加法モデル）
 │               ├── SVR（サポートベクター回帰）
 │               ├── Random Forest（決定木回帰）
 │               ├── LightGBM（高速な勾配ブースティング回帰）
 │               └── LSTM（長期記憶が可能な再帰型ニューラルネットワーク）
 │
 └──► 系統 C：【密度・距離・形状ベース】（波形を塊や別の空間に変換し、ボッチを探すルート）
       │
       ├── [ステップ1：分解・変換（前処理）] ── ※ここでの空間変換が必須条件
       │    ├── 【周波数領域への変換】
       │    │    ├── FT / DFT / FFT（（高速）フーリエ変換：全期間の周波数化）
       │    │    ├── STFT（短時間フーリエ変換：時間×周波数）
       │    │    ├── CWT / DWT（（離散）ウェーブレット変換：時間×周波数の高解像度化）
       │    │    └── WVD（ウィグナー・ヴィル分布：高次な時系列周波数分布）
       │    │
       │    └── 【構造・次元削減への変換】
       │         ├── SSA（奇異スペクトル解析：軌道行列によるパターン分離）
       │         ├── PAA（区分線形近似：時間軸のガサッとした圧縮）
       │         ├── SAX（Symbolic Aggregate approximation：波形の文字列化）
       │         └── 1D-PCA / カーネルPCA（主成分分析による特徴量化）
       │
       └── [ステップ2：判定・予測（本処理）] ── ※変換後の「特徴量の塊」に適用
            ├── 【機械学習（距離・密度ベース）】
            │    ├── k-NN（k近傍法：正常な塊からの距離測定）
            │    ├── LOF（Local Outlier Factor：周囲の密度からの孤立度）
            │    ├── One-Class SVM（正常データを取り囲む境界線の学習）
            │    ├── Isolation Forest（データを孤立させるまでの切り分け回数）
            │    └── MDS（多次元尺度構成法による異常プロット）
            │
            └── 【ディープラーニング（再構成誤差ベース）】
                 ├── Autoencoder（正常な波形だけを圧縮・復元できるように学習）
                 ├── VAE（変分オートエンコーダ：確率分布を介した復元学習）
                 └── LSTM-AE（時系列の形をそのまま圧縮・復元するオートエンコーダ）
